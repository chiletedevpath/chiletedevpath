# Documentación de Marca

Esta carpeta organiza los documentos base de Chilete DevPath.

Su objetivo es servir como fuente para construir la web de la marca, el portafolio, la comunicación pública y los criterios de publicación del ecosistema.

## Estructura

- `politicas/`: políticas formales de Chilete DevPath.

## Uso

Los documentos de esta carpeta deben mantenerse claros, versionados y alineados con la identidad de Chilete DevPath.

Cuando se cree la web de la marca, esta carpeta funcionará como referencia para secciones institucionales, criterios editoriales, uso responsable de IA, bienestar de la comunidad y otros recursos públicos.
