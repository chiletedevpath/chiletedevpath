# Assets de Marca

Esta carpeta contiene recursos visuales propios de Chilete DevPath y de Adrian Pisco.

## Autoría

Los archivos visuales de esta carpeta son recursos propios creados para la marca Chilete DevPath o fotografías personales autorizadas por Adrian Pisco para uso dentro de su ecosistema profesional.

No deben mezclarse con imágenes de terceros, personajes protegidos, logos ajenos o recursos descargados sin fuente clara.

## Uso recomendado

| Archivo | Uso sugerido |
|---|---|
| `portada-linkedin.png` | Portada principal del README y perfil profesional. |
| `marca-banner-principal.png` | Banner general para documentación, web o presentaciones. |
| `adrian-pisco-retrato.png` | Imagen personal para perfil, sección "sobre mí" o CV web. |
| `adrian-pisco-workstation.png` | Imagen contextual de trabajo y aprendizaje. |
| `marca-logo-sin-fondo.png` | Logo reutilizable sobre fondos claros u oscuros. |
| `marca-logo-fondo-blanco.png` | Logo sobre fondo blanco. |
| `marca-logo-fondo-oscuro.png` | Logo sobre fondo oscuro. |
| `marca-logo-texto-fondo.png` | Logo con texto para piezas de marca. |
| `marca-logo-texto-personal.png` | Logo con texto y presencia personal. |
| `web-captura-hero.png` | Captura de referencia para la futura web. |
| `web-captura-preloader.png` | Captura de referencia para experiencia visual de marca. |

## Criterio de reutilización

Estos assets pueden usarse en:

- README principal de Chilete DevPath;
- documentación de marca;
- futura web oficial;
- portafolio personal;
- materiales educativos propios;
- ejercicios donde tenga sentido usar identidad visual de Chilete DevPath.

Si se reutilizan en otro repositorio, debe copiarse solo el archivo necesario y mantener una nota de autoría o procedencia.

## Restricciones

- No usar estos archivos para simular una marca externa.
- No mezclarlos con recursos sin licencia o sin fuente.
- No modificar fotografías personales para contextos que puedan afectar la reputación profesional.
- No publicar versiones antiguas o duplicadas sin revisar su propósito.
